CRUD

CREATE - READ - UPDATE - DELETE

Este es el acronimo en ingles para crear, leer actualizar y borrar, que se usa para nombrar a las funciones basicas que debe tener un software de este tipo en bases de datos.


En esta oportunidad he creado este software CRUD con atributos basicos con la posibilidad de ingresar datos de personas como son "NOMBRE" Y "ROL"


A su vez dichos datos en esta pequeña base de datos pueden ser aparte de ingresados, modificados y tambien eliminados

Pasos a seguir

ATENCION

Si quieres descargar y probar


esta pequeña bade de datos


debes hacer lo siguiente :

Primero

Asegurate de tener instalado en tu computadora WampServer.

Luego tener iniciado todos los servicios y entrar a phpMyAdmin con usuario: root, sin contraseña y servidor MySQL

Luego de ello te haces un servidor llamado crud_nodejs_bd

Segundo

Abre esa carpeta y clona mi repositorio en GitHub, y con un editor de codigo abre lo que clonaste.

en la terminal de comandos de tu editor escribe node app y presiona ENTER. Aparecera: Servidor funcionando en http://localhost:5000 ¡Conectado a la BD MySQL!
